title: 'YouTube 导入'
description: '导入 YouTube 视频、播放列表和频道到 NotebookLM'
---

## 问题

YouTube 上有大量优质教程和讲座，想把多个视频的内容整理到 NotebookLM 做知识管理。但逐个复制链接太繁琐，播放列表和频道更是无从下手。

## 解决方案

<Steps>
  <Step title="获取 YouTube 链接">
    复制视频、播放列表或频道链接。也可以直接在 YouTube 页面打开扩展，自动检测当前链接。
  </Step>
  <Step title="粘贴到 Jetpack">
    点击 Jetpack 图标 → 切到「YouTube」tab → 粘贴链接 → 点击「查询」。
  </Step>
  <Step title="选择视频">
    播放列表/频道会显示视频列表，勾选要导入的视频。可选限制最近 N 个。
  </Step>
  <Step title="导入到 NotebookLM">
    点击导入，扩展将视频 URL 批量发送给 NotebookLM，由其原生解析 YouTube transcript。
  </Step>
</Steps>

## 支持的链接格式

| 类型 | 格式示例 | 说明 |
|---|---|---|
| 单个视频 | `youtube.com/watch?v=xxx`, `youtu.be/xxx` | 一键导入 |
| 播放列表 | `youtube.com/playlist?list=xxx` | 批量选择导入 |
| 频道 | `youtube.com/@username`, `youtube.com/channel/UCxxx` | 获取频道视频列表 |
| Shorts | `youtube.com/shorts/xxx` | 同单个视频 |
| 直播 | `youtube.com/live/xxx` | 同单个视频 |

## 导入数量限制

- 播放列表/频道模式下可设置「最近 N 个」
- 留空则获取全部视频
- 频道模式通过 RSS 获取最近 15 个，超过 15 个自动切换 InnerTube API

<Tip>
  NotebookLM 原生支持 YouTube 视频作为来源，Jetpack 的价值在于批量导入 — 一次性把整个播放列表或频道的视频添加为来源。
</Tip>

<Note>
  导入的是 YouTube 视频链接，NotebookLM 会自动获取视频的 transcript（字幕）。如果视频没有字幕，NotebookLM 可能无法解析内容。
</Note>
