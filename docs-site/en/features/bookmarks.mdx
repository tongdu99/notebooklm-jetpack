## The Problem

NotebookLM's free tier gives each notebook only **50 source slots**. Import 20 blog posts and you've already used up nearly half.

## The Solution

Bookmark multiple articles and merge them into a single PDF — taking up just **1 source slot**.

<Steps>
  <Step title="Bookmark articles">
    While browsing, click the Jetpack icon → click the "Bookmark" button to save the current page to your bookmarks.
  </Step>
  <Step title="Organize your bookmarks">
    View all bookmarked articles in the Bookmarks tab. Create different **Collections** to organize by project or topic.
  </Step>
  <Step title="Select and export">
    Check the articles you want to merge, then choose:
    - **Download PDF** — fetches article content, converts to Markdown, and merges into a warm book-style PDF (with cover, table of contents, and signature)
    - **Copy to clipboard** — click the 📋 icon next to the PDF button to copy Markdown-formatted content to your clipboard
  </Step>
  <Step title="Upload to NotebookLM">
    Drag the exported PDF into your NotebookLM notebook → **20 articles, just 1 source slot**.
  </Step>
</Steps>

## Collection Management

| Action | Description |
|---|---|
| Create collection | Click the ➕ button next to the collection name |
| Move bookmark | Use the "Move to..." dropdown on each bookmark |
| Delete collection | Bookmarks return to the default collection automatically |
| Bulk delete | Select multiple bookmarks → click the delete button |

<Tip>
  **Best practice**: Create collections by research topic (e.g., "AI Papers", "Product Design"), and export when you've accumulated 10-20 articles for maximum efficiency.
</Tip>

<Note>
  You can also click "Import to NotebookLM" to import articles individually without PDF merging. This works well when you have plenty of source slots available.
</Note>
