# Supercharge your NotebookLM 🚀

NotebookLM is the best AI knowledge tool, but getting content into it is painful.

**Jetpack fixes that.**

<CardGroup cols={3}>
  <Card title="One-click Import" icon="bolt">
    Web pages, Substack, WeChat, X.com articles, doc sites, YouTube, podcasts, AI chats — just paste a link
  </Card>
  <Card title="Break Source Limits" icon="layer-group">
    Merge 20 articles into 1 PDF that takes up just 1 source slot
  </Card>
  <Card title="Smart Rescue" icon="wrench">
    Auto-detect failed sources and batch-fix them in one click
  </Card>
</CardGroup>

## Supported Content

| Type | Sources |
|---|---|
| 🌐 Web Pages | Any URL, Substack, WeChat articles, X.com long-form |
| 📚 Doc Sites | Docusaurus, VitePress, MkDocs, GitBook, Mintlify, and 14+ frameworks |
| ▶️ YouTube | Videos, playlists, channels |
| 🎙️ Podcasts | Apple Podcasts, Xiaoyuzhou FM |
| 🤖 AI Chats | Claude, ChatGPT, Gemini |
| 📡 RSS | Any RSS/Atom feed |

- ✅ Completely free, no sign-up required

- ✅ [Open source](https://github.com/crazynomad/notebooklm-jetpack), fully auditable
- ✅ Bilingual UI (English & Chinese), auto-detected
